# Quality Review

Run this before scaling a batch or publishing a clip.

## Visual

- First frame communicates the topic or audience.
- Subject is framed correctly for the platform.
- No warped hands, mouth, face, or props.
- No extra unwanted people.
- No fake logos or unreadable generated text.
- Lighting and setting match the intended trust level.

## Script And Claims

- Hook is immediate.
- Promise is specific.
- No unsupported guarantee.
- No accidental implication of private-data access.
- CTA is clear and matches the next action.

## Audio

- Speech is intelligible on phone speakers.
- Music does not bury voice.
- Loudness is consistent across clips.
- Captions are readable if used.

## Platform Fit

- 9:16 for TikTok, Reels, Shorts unless there is a reason not to.
- Safe area respected for captions and CTA.
- Thumbnail frame is intentional.
- Native platform copy should be rewritten per platform instead of pasted everywhere.

## Batch Decision

Scale only if the test clip passes:

- right audience
- right presenter
- right first frame
- right claim
- right CTA
- no distracting artifacts

If any one of those fails, revise the spec before spending more credits.
