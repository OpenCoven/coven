# MEMORY.md taxonomy guidance

When promoting timeline snippets, prefer these long-term categories:

- Core preferences
- Communication style / boundaries
- Recurring schedule & routines
- Contacts
- Active projects
- Infrastructure / environment facts
- Lessons learned / mistakes to avoid

## Quality bar for durable memory

Promote only information that is:
1. likely to be reused,
2. stable enough to remain true,
3. actionable for future decisions.

Skip transient chatter and one-off noise.
